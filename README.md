# Yandex disk test

set your yadisk data in `yadisk.config.json`
```json
{
  "CLIENT_ID": "*your client id",
  "DEBUG_TOKEN": "*your debug token"
}
```

**ATTENTION!** All files are uploaded to the `Temp` folder, so you must first create this folder on your disk
